# GM Enviro Engineers — Premium Website

A modern, responsive, animated multi-page website for an Environmental Engineering & Consultancy company.

## ✨ Highlights
- 6 fully designed pages: Home, About, Services, Projects, Certifications, Contact
- Premium dark + eco-green theme with glassmorphism, gradients, particles & blobs
- Smooth scrolling with **Lenis**
- Scroll/entrance animations with **Anime.js** + **GSAP ScrollTrigger**
- **Swiper.js** testimonial slider
- Animated counters, magnetic buttons, custom cursor, animated mobile menu
- Fully responsive (mobile, tablet, laptop, ultrawide)
- Floating WhatsApp FAB + scroll progress bar

## 🚀 Run Locally
1. Open the project folder in **VS Code**.
2. Install the **Live Server** extension.
3. Right-click `index.html` → **Open with Live Server**.

> Internet connection required for fonts, CDN libraries, Unsplash imagery and the hero video.

## 🧩 Customize
- Edit colors / radii / shadows in `css/style.css` `:root`.
- Update content directly in each HTML page.
- Replace Unsplash URLs with your own assets in `assets/images/`.
